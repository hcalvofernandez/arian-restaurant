Features

  • Enable compatibility between your IP-based ESC/POS printer and Odoo Point of Sale.
  • After installing the app, printing POS tickets and receipts doesn't require an IoT Box.
  • Connect your POS printer to the same network as your Odoo POS.
  • Within the configuration menu, set the printer's IP address and port.
  • Enjoy streamlined and hassle-free printing without complex setups.
  • Easy to use.

Setup

  • Enter the printer IP address and port number inside the Point of Sale => Configuration => Settings
  • Enable the Automatic Receipt Printing configuration.

  • Click on validate to print using network printer
